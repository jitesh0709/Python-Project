# 📊 Exploratory Data Analysis (EDA) Projects

This repository contains a collection of Exploratory Data Analysis (EDA) projects completed using Python. Each project focuses on understanding a different real-world dataset through data cleaning, preprocessing, visualization, and extracting meaningful insights.

These projects helped me improve my Python and data analysis skills while working with datasets from different domains.

---

## 📁 Projects Included

### 🎬 Netflix Data Analysis

* Cleaned and explored the Netflix Movies & TV Shows dataset.
* Analyzed content by country, release year, ratings, genres, and duration.
* Created visualizations to identify trends and patterns.

---

### 🏏 IPL Match Data Analysis

* Explored IPL match data to understand team performance and player statistics.
* Analyzed toss decisions, match winners, top performers, and season trends.
* Built visualizations to compare teams and player performances.

---

### 🎌 Anime Dataset Analysis

* Explored an anime dataset containing ratings, popularity, genres, and episode information.
* Cleaned the data and identified trends in anime ratings and popularity.
* Used visualizations to better understand the dataset.

---

### 🌍 Countries Data Analysis

* Worked with a global countries dataset containing demographic, economic, and health-related information.
* Performed data cleaning and exploratory analysis.
* Compared countries using different indicators through charts and graphs.
